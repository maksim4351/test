How to Use This Website

  1. To start a test, choose one from the dashboard.
  2. Use the following controls during the tests:
    • Right Arrow (→): Move to the next screen or pattern.
    • Left Arrow (←): Go back to the previous screen or pattern.
    • Spacebar: Pause or resume automatic transitions.
    • Esc: Exit fullscreen mode.
  3. Follow the on-screen instructions for each test to ensure accurate results.
  4. To return to the main menu, press "Back to Dashboard."
